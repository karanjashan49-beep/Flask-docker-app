# 🚀 Flask Docker App

## 📌 About

This project demonstrates how to build and containerize a Python web application using Flask and Docker.

The application provides multiple web routes and runs inside a Docker container.

---

## 🛠 Technologies Used

- Python 3
- Flask
- Docker
- Linux
- Git
- GitHub

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python3 app.py
```

---

## 🐳 Build Docker Image

```bash
docker build -t flask-docker-app .
```

---

## 🐳 Run Docker Container

```bash
docker run -p 5000:5000 flask-docker-app
```
